# 摄影级（相机用）天文望远镜设计原理与思路

将天文望远镜与相机结合进行天文摄影（Astrophotography），其设计思路和目视观测有很大不同。目视观测追求的是放大倍率和人眼的舒适度，而天文摄影追求的是**集光力（光圈）、像场平整度、以及消像差能力**。简单来说，摄影天文望远镜本质上就是一个**超长焦、大光圈的定焦相机镜头**。

## 一、 天文摄影望远镜的核心原理

相机代替了人眼和目镜，直接放置在望远镜的**主焦点**位置（即主镜直接在相机传感器上成像，这称为“直焦摄影”Prime Focus Photography）。

1. **焦距与视场 (Focal Length & Field of View)**：
   焦距决定了目标在传感器上的大小（倍率）。焦距越长，视场越小，拍出的星系/行星越大。例如：拍深空星云通常需要 300mm-800mm 焦距；拍行星（木星、土星）通常需要 2000mm 以上焦距。
2. **焦比 (F-ratio = 焦距 / 口径)**：
   对应相机的光圈值 (F值)。F值越小（如 F/4, F/5），属于“快镜”，能在相同时间内收集更多光子，大幅缩短深空摄影的曝光时间。
3. **像场与平场 (Flat Field)**：
   普通望远镜在中心点成像锐利，但在边缘会产生彗差（星点变成拖尾的飞鸟状）或场曲（边缘失焦）。相机的传感器（尤其是全画幅）很大，因此摄影望远镜必须在整个传感器平面上提供平坦、无像差的像场。

## 二、 摄影级天文望远镜的经典设计

目前主流的天文摄影望远镜主要分为三大类阵营：

### 1. 复消色差折射镜 (APO Refractor) - 深空摄影的黄金标准
**原理**：利用两片或三片以上不同材质（包含特殊低色散ED玻璃、萤石等）的透镜组合，将红、绿、蓝三色光汇聚于同一点，消除色差。
**设计思路**：
- **物镜组**：通常为三片式（Triplet）空气隔开或胶合设计。
- **平场镜/减焦镜 (Flattener / Reducer)**：这是折射镜摄影**必不可少**的附件，安装在调焦座和相机之间。它能将球面像场拉平，同时往往还能缩短焦距（例如将F/7减焦到F/5.5），提高拍摄速度。
- **经典代表**：高桥(Takahashi) FSQ-106ED, 锐星(Sharpstar) ASKAR FRA系列。
- **优缺点**：星点极其锐利，反差高，几乎无需维护（无光轴烦恼）；但大口径成本呈指数级上升，通常在 60mm-130mm 口径。

### 2. 牛顿反射镜 (Newtonian Reflector) - 高性价比的大光圈“快镜”
**原理**：利用抛物面主镜汇聚光线，通过45度平面副镜将光线反射到侧面相机。无色差，大口径成本低。
**设计思路（摄影版牛反/碳纤维牛反）**：
- **快速焦比**：通常设计为 F/4 甚至 F/3.3，以极快速度收集深空光子。
- **大号副镜**：为了全画幅或APS-C传感器不产生四周暗角，必须使用比目视牛反更大的副镜（短轴尺寸通常在 60mm-90mm）。
- **彗差修正镜 (Coma Corrector)**：抛物面不可避免会产生彗差，摄影牛反**必须**在调焦座内插入 MPCC (Multi-Purpose Coma Corrector) 才能保证边缘星点为圆形。
- **高强度镜筒与调焦**：为了承载沉重的相机和制冷设备，必须使用碳纤维镜筒（膨胀系数极低，防止温度变化导致跑焦）和重型线性调焦座。
- **经典代表**：信达(Celestron) 小黑/大黑摄影版，Bresser F/4 系列，Sky-Watcher Quattro。

### 3. 折反射镜 (Catadioptric) / 施密特-卡塞格林 (SCT) & 锐星-卡塞格林 (RC) - 焦距王者
**原理**：结合了透镜（修正板）和反射镜，光路在镜筒内折叠三次，能在很短的镜筒内实现极长的焦距。
- **SCT (施密特-卡塞格林)**：
  前端有一片复杂形状的施密特修正板，主镜为球面镜，副镜为凸面镜。
  **设计思路**：常用于高分辨率的行星摄影（焦距动辄 2000mm+）。若要拍深空，可以使用 HyperStar 减焦平场镜降至 F/7，或者使用 Fastar/Hyperstar 系统直接拆下副镜，将相机装在前端，变成 F/2 的终极快镜。
- **RC (里奇-克莱琴 Ritchey-Chrétien)**：
  **设计思路**：双曲面主镜 + 双曲面副镜。这是专业天文台（如哈勃望远镜）的标配。天生无彗差，非常适合长焦深空摄影（星系摄影）。但光轴极难调节。
- **经典代表**：星特朗(Celestron) EdgeHD 系列 (SCT)，GSO RC8。

## 三、 相机连接方式 (Back Focus 与 M48/M42 接口)

自己设计或组装摄影望远镜时，最核心的工程挑战是 **“法兰距 / 截距 (Back Focus)”** 匹配：

1. **截距要求**：平场镜/彗差修正镜的最后一枚镜片，到相机CMOS传感器的距离必须**绝对精确**（通常标准是 55mm，允许误差在 1mm 以内），否则四周星点依然会拉线变形。
2. **连接件堆叠**：
   `调焦座 -> 平场镜/MPCC -> M48/M42螺纹延长圈 -> OAG(离轴导星器) -> 滤镜轮(EFW) -> 相机(DSLR或冷冻天文相机)`
3. 必须通过计算各个设备的厚度，购买精度合适的金属延长环（Spacer）组合来凑齐这 55mm。

## 总结：如果是DIY摄影望远镜的思路

如果您想 DIY 一台用于接单反相机的望远镜，最现实的方案是**“摄影版牛反”**：
1. **采购**：采购 150mm F/4 或 200mm F/4 的抛物面主镜（1000元左右），以及至少 63mm 短轴的副镜。
2. **结构**：必须使用高刚性管材（如 5mm 厚的铝管或碳纤维管，不可用PVC，因为PVC受温度影响大，且容易弯曲导致光轴偏移）。
3. **调焦座**：购买 2英寸双速无齿调焦座（Crayford Focuser），打印件强度不足以支撑单反相机的重量。
4. **核心配件**：购买一个标准的 2英寸牛反彗差修正镜（MPCC，约800元）。
5. **组装与打孔**：精确计算光路，确保调焦座拉出到一半时，相机的传感器刚好落在主焦点上，并且加上 MPCC 后截距正好为 55mm。
